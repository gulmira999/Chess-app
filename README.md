# ♟ Chess App

Шахматная игра против ИИ, созданная на React.

## Функционал
- Игра против ИИ (алгоритм Minimax с alpha-beta отсечением)
- Полные правила: рокировка, взятие на проходе, превращение пешки
- Выбор цвета (белые/чёрные)
- Определение мата, пата и шаха
- Красивый тёмный дизайн

## Технологии
- React
- JavaScript
- Vercel (деплой)
